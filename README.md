## 📌 Overview

**Here-and-Now-AI** is a modular framework for building real-time, context-aware AI interaction engines. It provides an extensible foundation for developing intelligent agents and automated AI workflows — designed so developers can rapidly prototype and deploy sophisticated AI-powered applications without rebuilding core infrastructure from scratch.

The repo contains **9 independent sub-projects** (`proj1`–`proj8`, `proj11`), each exploring different aspects of real-time AI interaction, backed by a unified application layer (`app.py`) and UI system (`ui1.py`).

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🧠 Context-Aware Workflows | AI pipelines that maintain and use session context across interactions |
| ⚡ Real-Time AI Pipelines | Low-latency processing for live AI response generation |
| 🤖 Intelligent Agents | Autonomous agents capable of multi-step task completion |
| 🔌 Extensible Plugin Architecture | Modular `proj*` structure — add new AI capabilities without touching core |
| 🎨 Branding System | `branding.json` for consistent AI persona theming across projects |
| 🖥️ UI Layer | `ui1.py` — unified interface layer across all sub-projects |

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/Bhogi123-T/Here-and-Now-AI.git
cd Here-and-Now-AI

# Install dependencies
pip install -r requirements.txt   # if present

# Run main application
python app.py

# Run with UI layer
python ui1.py
```

## 🔄 Architecture

```
User Input
    ↓
Context Manager (session state)
    ↓
AI Pipeline (real-time processing)
    ↓
Agent Decision Engine
    ↓
Plugin / Module (proj1–proj11)
    ↓
Response Generation
    ↓
UI Layer (ui1.py)
```
